# Flatbuffers Haxe
An implementation of Google's FlatBuffers in Haxe. 

## How to install

```bash
  haxelib git flatbuffers https://github.com/troyedwardsjr/flatbuffers-haxe.git
```

Current targets: 
JS, 
C++/Neko, 
Android(Java) (Untested, but should work), 
AS3 (Untested, but should work),
ect.
